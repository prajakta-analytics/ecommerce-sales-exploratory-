## E-Commerce Sales Exploratory Dashboard

An interactive dashboard designed to analyze e-commerce sales data and uncover key insights such as sales trends, product performance, and business metrics.

### Key Highlights
- Sales and profit analysis
- Category and product performance
- Trend-based insights using visual dashboards

### Tools Used
- Power BI / Excel / Python (EDA)
